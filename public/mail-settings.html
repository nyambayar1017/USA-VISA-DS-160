How to get an app password

  1. Sign into the mailbox at mail.google.com
  2. Open myaccount.google.com/apppasswords (turn on 2-Step Verification first if prompted)
  3. Name it TravelX backoffice and click Create
  4. Copy the 16-character code (with or without spaces — both work) and paste it below

Where do Spam, Promotions and Social emails go?

Gmail's Promotions, Social and Updates tabs are not separate folders — they all live inside Inbox. So they show up here automatically alongside your normal mail.

Spam is a separate Gmail folder; we don't sync it on purpose so the back-office stays clean. Open Gmail's Spam folder directly if a real email got filtered there, then mark it "Not spam" so future ones land in Inbox.
